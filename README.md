# 📝 To Do List (HTML, CSS e JavaScript)

## 📌 Descrição do Projeto 

Este projeto consiste em uma aplicação simples de **lista de tarefas (To Do List)** desenvolvida utilizando apenas **HTML, CSS e JavaScript puro (Vanilla JS)**.

O objetivo é permitir que o usuário:

* Adicione novas tarefas
* Defina uma prioridade (Alta, Média ou Baixa)
* Visualize as tarefas na tela
* Marque tarefas como concluídas

---

## 🚀 Funcionalidades

* ✅ Adicionar tarefas dinamicamente
* 🎯 Definir prioridade:

  * 🔴 Alta
  * 🟡 Média
  * 🟢 Baixa
* ✔️ Marcar tarefas como concluídas
* 🎨 Mudança de estilo conforme prioridade
* ♿ Melhorias de acessibilidade (labels, estrutura semântica)

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** → Estrutura da aplicação
* **CSS3** → Estilização e responsividade
* **JavaScript (Vanilla)** → Lógica e interatividade
